### 275. prac - Using Helm to deploy a chart
- 헬름 명령을 통해서 ArtifactHub.io에서 `consul` 헬름 차트 검색
```bash
controlplane ~ ➜ helm search hub consul
```

- `bitnami` 레포 추가
```bash
controlplane ~ ➜  helm repo add bitnami https://charts.bitnami.com/bitnami
"bitnami" has been added to your repositories
```

- 현재 헬름 레포 확인
```bash
controlplane ~ ➜  helm repo list
NAME    URL                               
bitnami https://charts.bitnami.com/bitnami
```

- 현재 레포에서 `wordpress` 차트 검색
```bash
controlplane ~ ➜  helm search repo wordpress
NAME                    CHART VERSION   APP VERSION     DESCRIPTION                                       
bitnami/wordpress       25.0.25         6.8.2           WordPress is the world's most popular blogging ...
bitnami/wordpress-intel 2.1.31          6.1.1           DEPRECATED WordPress for Intel is the most popu...
```

- 현재 헬름 레포 목록 확인
```bash
controlplane ~ ➜  helm repo list
NAME            URL                                                 
bitnami         https://charts.bitnami.com/bitnami                  
puppet          https://puppetlabs.github.io/puppetserver-helm-chart
hashicorp       https://helm.releases.hashicorp.com  
```

- `bitnami` 레포에서 `apache` 차트를 배포
```bash
controlplane ~ ✖ helm install amaze-surf bitnami/apache
```

- 배포한 `apache`의 앱 버전 확인
```bash
controlplane ~ ➜  helm list
NAME            NAMESPACE       REVISION        UPDATED                                 STATUS          CHART           APP VERSION
amaze-surf      default         1               2025-08-16 13:42:12.777562089 +0000 UTC deployed        apache-11.3.2   2.4.63 
```

- 현재 배포된 릴리즈 확인
```bash
controlplane ~ ➜  helm list
```

- `happy-browse` 릴리즈 제거
```bash
controlplane ~ ➜  helm uninstall happy-browse
```
