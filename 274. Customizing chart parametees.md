### 274. Customizing chart parametees
헬름 차트의 `values.yaml` 파일의 매개변수 값을 통해, 배포할 애플리케이션의 설정을 변경할 수 있다.

- 명령을 통해 `values.yaml` 값을 변경
	- `--set` 옵션을 통해, `values.yaml`의 값을 명령 줄에서 바로 변경이 가능하다. 여러 번 사용하는 것도 가능하다.
```bash
helm install --set wordpressBlogName="Helm Tutorials" my-release bitnami/wordpress
```

- 명령을 통해 다른 파일을 `values.yaml` 파일로 사용
```bash
helm install --value custom-values.yaml my-release bitnami.wordpress
```

- 차트 가져오기
```bash
helm pull bitnami/wordpress
```

- `helm pull`을 통해 차트를 가져오면, 압축된 형태로 가져온다. 이를 압축 풀기
```bash
helm pull --untar bitnami/wordpress
```

- 로컬로 다운로드 받은 helm 차트로 애플리케이션 배포
```bash
# helm insatll my-release <HELM_CHART_PATH>
helm insatll my-release ./wordpress
```
